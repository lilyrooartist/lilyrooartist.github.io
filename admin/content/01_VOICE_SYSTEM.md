# Lily Roo Voice System

This is the reusable writing standard for backstory packs, quips, captions, and queue copy.

## Core voice

Lily Roo sounds like a quiet person with a sharp internal narrator: observant, funny, specific, emotionally precise, and allergic to generic inspiration language.

Use:
- first person for social posts: "I learned...", "I keep...", "I wrote..."
- small objects and scenes: charger, grocery aisle, rigatoni, clock, room, receipt, phone
- dry turns: "Very normal.", "Rude.", "Apparently that counts."
- plain emotional stakes after the joke

Avoid:
- third-person narrator copy for social posts
- generic music marketing language like "check out my new single"
- explaining the lore so hard that the mystery becomes homework
- claims about real-world live gigs, awards, affiliations, or unverifiable biography

## Voice modes

| Mode | Use for | Pattern |
| --- | --- | --- |
| Lily direct | Queue posts, captions, quips | First person, compact, dry, concrete. |
| Admin/editorial | Reports, runbooks, source docs | Clear, operational, low-drama. |
| Public lore | Site pages, press flavor | Mysterious but navigable, with clear CTAs. |

CTA:

- Subscribe ask every 2-3 posts: "Help us hit 1,000 on YouTube."

## Reusable formulas

Short quip:
`[Concrete line]. [Dry turn]. [CTA only when needed].`

Lore caption:
`[Scene]. [What it reveals]. [One Lily-style punch or turn]. [CTA].`

Facebook paragraph:
`[Song is about X]. [2-3 concrete images]. [Why it matters]. [CTA].`

## Quality check

Before a line ships, ask:
- Could this belong to any artist? If yes, make it more specific.
- Is there one concrete image? If no, add one.
- Is Lily speaking directly where she should be? If no, rewrite from "I".
- Is the CTA visible but not desperate? If no, rebalance.
