# I Learned It All in Fifteen Seconds Assets

Active first-album assets for Lily Roo.

- `art/`: remastered 3000 x 3000 album and track artwork.
- `audio/`: current local WAV masters used for the DistroKid album upload.
- `video-canvases/`: 1920 x 1080 art canvases used to render replacement YouTube videos.
- `video/`: rendered replacement YouTube masters uploaded on 2026-06-07.
- `album-assets.json`: track-level manifest with current YouTube URLs and archived original URLs.

No separate DistroKid-enhanced downloadable masters were found locally during this cleanup, so the replacement videos use the DistroKid-upload WAV masters from the first-album package.
