# Lily Roo site map + conventions

## Pages
- `/` (index.html): hero + featured release + primary CTAs + embed + broadcast windows
- `/music.html`: discography hub
- `/press.html`: press/EPK
- `/contact.html`: Google Form contact

## Conventions
- Root-relative links (`/music.html`).
- Privacy: YouTube embeds via `youtube-nocookie.com`.
- OG image should be absolute URL under `https://lilyroo.com/assets/...`.
