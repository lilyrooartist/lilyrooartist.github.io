# Analytics / Feedback Loop

## KPIs (minimum viable)
- Play clicks / video opens
- Mailing list signups
- YouTube: views first 24–72h

## Operating rules
- Recommend small changes with a hypothesis.
- Change one thing at a time when possible.

## Data sources
- GitHub Pages deploy health (spot checks)
- YouTube public stats (manual/visible)
- Mailing list sheet row counts
