# Social Posting Ops (Drafts + Queue)

## Operating model
- Produce drafts and a posting schedule.
- Only post automatically when explicit platform integrations exist.

## Inputs (ask if missing)
- Asset: link (YouTube), cover image, 1–2 key lines.
- Goal: play-through vs. email signup vs. follow.

## Outputs
- 1 hook per platform (IG/TikTok/X/YouTube Community)
- 2 caption variants (short + lore)
- 8–12 hashtags (TikTok/IG)
- CTA variants
